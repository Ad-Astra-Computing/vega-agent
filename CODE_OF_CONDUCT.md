# Code of Conduct

This project adopts the Contributor Covenant, version 2.1, as its code of
conduct. The full text is at
https://www.contributor-covenant.org/version/2/1/code_of_conduct/.

In short: treat everyone with respect, assume good faith, and keep discussion
focused on the work. Behavior that harasses, demeans, or excludes others is not
welcome.

## Enforcement

Report unacceptable behavior to jason@adastracomputing.com. Reports are handled
confidentially. Maintainers may remove comments, commits, issues, and pull
requests that violate this code, and may ban repeat or severe offenders.
